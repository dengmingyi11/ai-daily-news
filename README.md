# 🤖 AI 行业日报 - 飞书推送版

每天北京时间 9:30 自动推送 AI 行业日报到飞书，国内稳定可靠。

---

## 🚀 部署步骤

### 1️⃣ 上传文件到 GitHub

访问：https://github.com/dengmingyi11/ai-daily-news

上传所有文件

### 2️⃣ 配置 1 个 Secret

**Settings → Secrets → Actions → New repository secret**

| Name | Value |
|------|-------|
| `FEISHU_WEBHOOK_URL` | `https://open.feishu.cn/open-apis/bot/v2/hook/e55697f5-509b-49be-95a5-629068792d1d` |

### 3️⃣ 启用 Actions 权限

**Settings → Actions → General**
- 选择 **Read and write permissions**
- 点击 **Save**

### 4️⃣ 测试

**Actions → AI Daily News → Run workflow**

---

## ✅ 完成！

每天北京时间 **9:30**，AI 日报自动推送到你的飞书！
