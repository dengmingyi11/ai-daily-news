## 4️⃣ 修改推送时间

**位置：** `.github/workflows/ai-daily-news.yml` 第 6 行

**当前配置：**
```yaml
schedule:
  - cron: '30 1 * * *'  # UTC 时间 01:30 = 北京时间 09:30
```

**常用时间配置：**

| 北京时间 | cron 表达式 |
|----------|-------------|
| 08:00 | `0 0 * * *` |
| 09:30 | `30 1 * * *` |
| 12:00 | `0 4 * * *` |
| 18:00 | `0 10 * * *` |
| 21:00 | `0 13 * * *` |

## 5️⃣ 修改飞书消息格式

**位置：** `scripts/generate_report.py` 第 120-161 行

**当前配置：** 使用 `text` 类型，简单文本格式

**修改为卡片消息（更美观）：**
```python
def send_feishu(report: str, date_str: str):
    """发送飞书卡片消息"""
    import requests
    
    webhook_url = os.environ.get('FEISHU_WEBHOOK_URL')
    
    if not webhook_url:
        print("⚠️ 飞书 Webhook URL 未配置，跳过推送")
        return False
    
    try:
        # 使用卡片消息
        data = {
            "msg_type": "interactive",
            "card": {
                "header": {
                    "title": {
                        "tag": "plain_text",
                        "content": f"🤖 AI 行业日报 - {date_str}"
                    },
                    "template": "blue"
                },
                "elements": [
                    {
                        "tag": "markdown",
                        "content": report
                    },
                    {
                        "tag": "action",
                        "actions": [
                            {
                                "tag": "button",
                                "text": {
                                    "tag": "plain_text",
                                    "content": "查看更多"
                                },
                                "type": "primary"
                            }
                        ]
                    }
                ]
            }
        }
        
        headers = {'Content-Type': 'application/json'}
        response = requests.post(webhook_url, json=data, headers=headers, timeout=10)
        result = response.json()
        
        if response.status_code == 200 and result.get('code') == 0:
            print(f"✅ 飞书推送成功！")
            return True
        else:
            print(f"❌ 飞书推送失败: {result.get('msg', '未知错误')}")
            return False
        
    except Exception as e:
        print(f"❌ 飞书推送失败: {e}")
        return False
```

## 🔧 修改步骤

### 第一步：克隆或下载仓库
```bash
git clone https://github.com/dengmingyi11/ai-daily-news.git
```


访问 GitHub Actions 页面，手动运行一次查看效果。
