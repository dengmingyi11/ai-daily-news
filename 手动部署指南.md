## 方法一：使用 GitHub CLI（推荐）

### 1. 安装 GitHub CLI
下载地址：https://cli.github.com/

### 2. 登录 GitHub
```bash
gh auth login
```

### 3. 推送代码
```bash
cd /path/to/ai-daily-news
git push -u origin main
```

---

## 方法二：使用 HTTPS + Personal Access Token

### 1. 创建 Personal Access Token
1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 `repo` 权限
4. 生成并复制 Token

### 2. 推送代码
```bash
cd /path/to/ai-daily-news
git remote set-url origin https://<TOKEN>@github.com/dengmingyi11/ai-daily-news.git
git push -u origin main
```

## 方法三：手动上传文件（最简单）

### 1. 访问你的仓库
https://github.com/dengmingyi11/ai-daily-news

### 2. 点击 "uploading an existing file"

### 3. 将以下文件拖入上传：
- `.github/` 文件夹
- `scripts/` 文件夹
- `reports/` 文件夹
- `README.md`
- `requirements.txt`

### 4. 点击 "Commit changes"
