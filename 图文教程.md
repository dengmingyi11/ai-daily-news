## 第一步：创建 GitHub 仓库

### 1.1 登录 GitHub

访问 **https://github.com** 并登录你的账号

### 1.2 创建新仓库

点击右上角 **+** 号，选择 **New repository**

## ❓ 常见问题

### Q1: 没有收到邮件？

1. 检查垃圾邮件文件夹
2. 查看 Actions 运行日志是否有错误
3. 确认 Secrets 配置正确

### Q2: Actions 运行失败？

1. 检查文件是否全部上传
2. 确认 Workflow permissions 已设置为 Read and write
3. 查看错误日志定位问题

### Q3: 想修改推送时间？

编辑 `.github/workflows/ai-daily-news.yml`：

```yaml
schedule:
  - cron: '30 1 * * *'  # 修改这个时间
```

| 北京时间 | Cron 表达式 |
|----------|-------------|
| 08:00 | `0 0 * * *` |
| 09:30 | `30 1 * * *` |
| 12:00 | `0 4 * * *` |
| 18:00 | `0 10 * * *` |
